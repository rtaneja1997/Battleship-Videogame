# Battleship

This project is an OCaml implementation of Battleship.

Instructions for compiling and running the game:

- To compile and play the game, run make play.

- To compile and run the test suite, run make or make test.

- To compile the code without doing anything else, run make compile.

- To remove the compiled files, run make clean.

- To create a zip folder containing only the code and this README, run make zip.
